# TWC Upgrade Rationale

The current repository already supports realistic GTA-like FS placement through ISED data, 3GPP-style UMi/O2I large-scale modeling, and classical WMMSE with average FS interference control.

The new package upgrades the work in five directions:
1. wideband NR grid + frequency selectivity,
2. hybrid cognitive + spatial coexistence,
3. risk-aware unfolded optimization,
4. explicit tone-group validity and user-weight analysis,
5. reference-geometry and TWC-style figure generation.
